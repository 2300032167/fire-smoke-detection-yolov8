# 🔥 Fire and Smoke Detection System

## 🚀 Overview
A real-time fire and smoke detection system using YOLOv8 and OpenCV with automated alert mechanisms.

## ✨ Features
- Real-time detection via webcam/video
- Automated SMS alerts
- Email notification system
- Alarm sound trigger
- Low-latency detection

## 🛠 Tech Stack
- Python
- OpenCV
- YOLOv8
- Twilio API (SMS)
- SMTP (Email)

## ▶️ How to Run
1. Install dependencies
2. Run:
   python webcam_detect.py
3. System detects fire and triggers alerts
